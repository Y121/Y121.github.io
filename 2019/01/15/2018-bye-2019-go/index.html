2018 bye,2019 go

发表于 2019-01-15 | 分类于 生活

现在

　　今天已经是2019年1月15日，距离到新年已经过了半个月，这个月时间基本是颓废度过的。元旦假期结束后的第一天上班，在电话里被公司通知辞退，什么原因呢，与公司发展不匹配？都是些老掉牙的借口，其实心里也有数，我这是被同事给黑了？当时除了有第一次被辞退的紧张，也说不上有什么落差，毕竟这一年来对公司都有不满，只是在拖延时间而已，所以接到电话的时候可能还觉得解脱和开心，终于结束了！第二天就是部门老大的谈话，毕竟在这里待了也有两年，从言语中也能听出她的无奈，可能她也有点自责，不知道，反正觉得她真是个天使，做她的下属很幸运!事情到了这个地步也不能做什么，她一直在安慰我，说被辞退不是我的错，我个人没有问题，不要受影响，还拿她的人生经历来开导我（其实我当时心理还真没什么影响，顶多觉得委屈吧），第二天又请我们（还有一个同事）吃饭，又是一轮新的激励与开导😂。然后她跟hr说工作交接要一个星期，然后等hr走后，她就让我也别来了，直接补卡就行，让我拿满一个月的工资，真的是一个天使❤。离开公司后，在租房肥宅了一个星期，周末买票回家，然后又肥宅到今天。。。回看在公司这两年，其实在技术与知识上没有什么进步，当初进公司会的一些东西反而忘记了，其实早就应该离开，只是自己安于现状，到现在这个地步。刚开始一年还好，2018这一年除了大屏展示，没有接触到新的技术，自己在这里也没有主动的心思，所以还是有问题吧，应该往前看，往上看，主动学习才是对的！

2018

　　回看2018年立的flag：
　　
　　第一条完成的马马虎虎，第二条没有完成，vue和es6都没有完整的学习，重新学习js更不用想了，上半年一直出差，学过的也忘记了，又没有练习，下半年不用出差也没有心思学习，直到2018最后一个月回看这些flag有些心慌，执行计划学习了半个多月，有些进步，但也没有学完。第三条尽量放下手机也没有做到，感觉有些近视了。第四条多看书多看报也没有做到，买了很多书，看了一半吧，还有一半还是新书。第五条多运动，只是报了一个瑜伽班，下半年有定时练瑜伽，这个还是很舒服的。第六条化妆，还能看吧。回看2018心好累，自制力差，惰性大，还有拖延症，毛病一大堆。但是呢，再说回来，虽然立下的flag没有都完成，但对自己还是有一点影响，应该说有一点束缚与压力吧，我觉得很好，总还是动手了，不然一年又是碌碌无为，什么都没做。2018跟同事一起去重庆玩了几天，很舒服，到处看看扩宽眼界总没错，心情也放松。

2019

　　现在2019已经过去半个月了，今年情况特殊，放假比较早，在家里可以有一个月左右的假期，家里真的太舒服了，舒服的什么都不想做，想就这样闲适的度过这一个月，但是不行！虽然不知道能不能做到，但是也给这一个月定下两个目标吧。多看书少玩手机，这一个月把《白夜行》看完。学习vue，为年后找工作做好准备，vue视频要看完，写出一个demo。
　　接下来2019，我希望我自己比起2018年要有所改变，有些毛病真的自己看着都已经到了厌倦的程度。
　　1.　改掉拖延症！真的不要再拖了，不论是自己给自己安排的事情，还是别人安排的事情，不管事情多大多小，第一反应就要去做，不要给自己做借口来推迟。这个毛病真的很讨厌！！！
　　2.　多看书，少玩手机。尽量让自己离开手机，看书是很舒服的一件事情，希望至少看十本书。
　　3.　坚持学习，能有所成。
　　4.　继续练习瑜伽。
　　5.　出去玩一趟吧，厦门、三亚或北海。
　　6.　学会画眼影。
　　时间过得越来越快，就怕以后回忆现在都想不起来到底做了什么，行动吧，啦啦啦:simple_smile:！
